# TextUtils

TextUtils is a React-based text utility application that helps users analyze and transform text efficiently. It provides multiple text manipulation features along with customizable themes and a clean user interface.

## Features

* Convert text to Uppercase
* Convert text to Lowercase
* Convert text to Alternate Case
* Remove Extra Spaces
* Clear Text
* Copy Text to Clipboard
* Download Text as a `.txt` file
* Word Count
* Character Count
* Reading Time Estimation
* Live Text Preview
* Multiple Color Themes
* Responsive Design
* Home, About, and Contact Pages
* Real-time Alert Notifications

## Tech Stack

* React.js
* React Router DOM
* Bootstrap 5
* JavaScript (ES6+)
* HTML5
* CSS3

## Installation

Clone the repository:

```bash
git clone https://github.com/Khushi27-agg/textutils.git
```

Navigate to the project folder:

```bash
cd textutils
```

Install dependencies:

```bash
npm install
```

Run the application:

```bash
npm start
```

Open:

```text
http://localhost:3000
```
